typedef 和 define的区别

define [标识符] [被标识符替换的字符串]
eg.

      #define ON   1   
      #define OFF  0

define是预处理指令
在预处理的时候标识符替换对应的字符串 写的时候要以#号开头
(作用就像是做一个文本的替换)

typedef [数据类型] [字符串]
typedef是c语言的一个关键字 作用是为已有的数据类型设定一个新的名字 在写的时候要以分号结尾
eg.

     typedef unsigned char u8;
     typedef unsigned int u16;
